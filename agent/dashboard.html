Q-Learning — Off-Policy

Q(s,a) <- Q(s,a) + α · [r + γ · max Q(s',a') − Q(s,a)] ↑ best possible next action

Off-policy: updates toward the optimal next action even if the agent does not take it. Learns the greedy policy regardless of exploration — tends to converge faster. Ideal when finding the optimal path is the priority.

SARSA — On-Policy

Q(s,a) <- Q(s,a) + α · [r + γ · Q(s', a') − Q(s,a)] ↑ actual next action taken

On-policy: updates using the action the agent will actually take, including exploratory moves. Learns a safer, more conservative policy. Slightly slower to converge but more robust in stochastic environments.
