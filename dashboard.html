Q-Learning (Off-Policy)

Q(s,a) ← Q(s,a) + α · [r + γ · max Q(s',a') − Q(s,a)] ↑ best possible next action

Off-policy: updates toward the optimal action even if the agent doesn't take it. Learns the greedy policy regardless of exploration. Tends to converge faster and find higher-reward paths. Suitable for environments where optimal performance is the priority.

SARSA (On-Policy)

Q(s,a) ← Q(s,a) + α · [r + γ · Q(s', a') − Q(s,a)] ↑ actual next action taken

On-policy: updates using the action the agent will actually take next — including exploratory moves. Learns a safer, more conservative policy. Slightly slower convergence but more robust in stochastic or risky environments. Both reach 100% success here.
